Ever wished you could use your smartphone or tablet as a drawing pad for your computer? This project allows you to turn any touch-screen device into a remote input controller for your PC.

## üöÄ Getting Started

Follow these steps to get your mobile drawing pad up and running.

### Prerequisites
* **Python 3.11.5**
* A PC and a mobile device connected to the **same Wi-Fi network**.

### Installation

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/Jubayer-Hossain-JB/Wireless-Draw-Pad.git](https://github.com/Jubayer-Hossain-JB/Wireless-Draw-Pad.git)
    cd Wireless-Draw-Pad
    ```

2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

### How to Use

1.  **Run the Application**: Execute the main script from your PC:
    ```bash
    python main.py
    ```
2.  **Get the Link**: A GUI will appear on your screen showing a **web link**.
3.  **Connect your Device**: Open the web browser on your mobile phone or tablet and enter that link.
4.  **Control your PC**: Enjoy controlling your PC and drawing via your mobile touch screen!
